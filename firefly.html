Glowing Serenity

Glowing Serenity is an interactive ambient game developed using Unity. Enjoy the peaceful night scene and bring a touch of natural glow~

1. For any device with any browser

You can find the online playable version here: https://lekaaa.itch.io/firefly

  • It works well on any browser;
  • Make sure to unmute your device to hear the sound effects;
  • Fullscreen is recommended.

2. For Wallpaper

If you have Wallpaper Engine (wallpaperengine.io), try subscribing and using it as your dynamic desktop wallpaper: https://steamcommunity.com/sharedfiles/filedetails/?id=3434517925

3. For Windows or Mac

More generally, you can run the executable program on Windows or Mac:

  • Windows: Click on Firefly.exe in the Windows folder to run it on Windows.

    • Download: GlowingSerenity_Windows.rar

  • Mac: Open the Mac folder and run the .app file to launch it on Mac.

    • Download: GlowingSerenity_Mac.rar
